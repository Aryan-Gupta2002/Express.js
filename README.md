# 🚀 Backend Learning Projects Repository

This repository contains a collection of backend and full-stack practice projects built while learning **Node.js, Express, APIs, and authentication** concepts.

Each folder represents a specific topic or weekly learning module.

---

## 📂 Project Structure

```
.
├── fetchAPI
├── frontend backend Sign-In
├── hospital-backend
├── middleware
├── week-4-todo-app
├── week-5-CORS
├── week-6-authentication
```

---

## 📌 Projects Overview

### 🔹 fetchAPI

* Demonstrates background API requests using `fetch`
* Covers async operations and handling responses

---

### 🔹 frontend backend Sign-In

* Basic Sign-In page with frontend + backend integration
* Focus on request handling (not styled with CSS)

---

### 🔹 hospital-backend

* REST API built using Express
* Supports:

  * `GET`
  * `POST`
  * `PUT`
  * `DELETE`
* Simulates hospital data management

---

### 🔹 middleware

* Custom middleware implementation
* Logs:

  * Request method
  * URL
  * Host
  * Timestamp

---

### 🔹 week-4-todo-app

* Todo application with backend support
* Demonstrates:

  * Query parameters
  * Server-side data handling

---

### 🔹 week-5-CORS

* Handles frontend-backend communication
* Implements **CORS (Cross-Origin Resource Sharing)**

---

### 🔹 week-6-authentication

* Authentication system with:

  * Sign-in
  * Sign-up
  * Protected routes
* Uses authentication middleware

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* JavaScript (ES6+)
* REST APIs

## ▶️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies (inside each project)

```bash
npm install
```

### 3. Run the project

```bash
node index.js
```

or (if using nodemon)

```bash
nodemon index.js
```
